The idea is to implement objects prioritization problem. In this problem the robot is programmed for selecting and picking objects from a set of available objects on the map based on their priority. Each object is assigned a rank that determines relative significance. On top of that optimize the minimum time and sending control signal to the existing MPC problem to reach desired coordinates on the map.
